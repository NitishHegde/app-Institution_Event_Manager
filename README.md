The Institution Event Management Portal is a comprehensive web application designed to streamline the planning, management, and participation of college events. It provides dedicated interfaces for students, faculty, coordinators, and administrators, enabling efficient event creation, registration, attendance tracking, result publication, and analytical reporting.

Features
 Role-based authentication (Student, Staff, Admin)
 Event creation and management
 Individual and team event registration
 Faculty and student coordinator management
 Attendance management
 Result publication and accomplishments tracking
 Interactive analytics dashboard
 Poster and rulebook PDF management
 Student profile search
 Secure JWT authentication
 PostgreSQL database with soft-delete support
 Designed for Azure Blob Storage integration

##
Tech Stack:
Frontend: React.js, React Router
Backend: Node.js, Express.js
Database: PostgreSQL
Authentication: JWT
File Storage: Local Storage (Development), Azure Blob Storage (Production)

This project follows a modular architecture with separate layers for controllers, services, repositories, and database access, making it scalable and maintainable for real-world academic environments.
